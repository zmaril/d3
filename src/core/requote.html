requote.js
¶

This little guy escapes a certain set of characters that are used often in CSS classes. Used to help selections know how to deal with CSS basically. Check the tests and introduction of the function to know more. 

d3.requote = function(s) {
  return s.replace(d3_requote_re, "\\$&");
};
¶

A small beast of a regex. 

var d3_requote_re = /[\\\^\$\*\+\?\|\[\]\(\)\.\{\}]/g;
